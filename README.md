# MERN Todo App

A full-stack Todo application built using the MERN stack.

## Features
- User Registration & Login
- JWT Authentication
- Create, Read, Update, Delete Todos
- Complete/Undo Todos
- Search and Filter
- Dark Mode
- Toast Notifications

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
